The Course: Regression Analysis

Before I talk about the infrastructure, here is a brief description of the course to give you some context:

Regression Analysis is a second-semester undergraduate course focused on analyzing multivariate data using regression models. Because the course is early in the undergraduate curriculum, the primary focus is on application rather than the mathematical theory. A more detailed description of the course is at the top of the course website.

There were 90 students in the course in Spring 2020, covering all years - freshmen to senior. The course is designed to include a lecture 2X a week and a lab with 30 students in each lab section.
